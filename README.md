# 知识猎手 (Knowledge Hunter) 🎯

> 主动出击，精准捕获有价值的内容，转化为你的知识资产

自动监控多平台内容更新，将音频/视频转成文字稿，并生成每日简报推送

## 一句话介绍

**免费、自动、聪明** —— 让AI帮你把所有关注的博主内容变成可搜索的文字库，并主动推荐值得看的内容

## 核心功能 (v1.1.0)

### 🚀 从被动存储 → 主动推送价值

| 功能 | 说明 |
|------|------|
| **观点提取** | 自动从转录稿提取核心观点、金句亮点 |
| **内容评分** | 基于观点质量、信息密度、原创性打分 |
| **冲突检测** | 发现不同来源的观点冲突 |
| **每日简报** | 每天9点推送精选内容简报 |

### 📦 基础监控功能

- 🎙️ **小宇宙播客** → 123分钟音频，20分钟转录完成
- 📺 **YouTube视频** → 优先提取官方字幕，秒出结果
- 📱 **B站视频** → 自动监控UP主更新
- 🐦 **Twitter推文** → 实时追踪行业动态

## 技术亮点

- ✅ **完全免费**：使用Whisper开源模型，无API费用
- ✅ **AI赋能**：LLM智能提取观点，告别信息过载
- ✅ **每日简报**：自动生成内容简报，推送飞书
- ✅ **观点冲突**：发现不同博主的观点差异
