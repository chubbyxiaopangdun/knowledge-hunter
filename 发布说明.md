## 更新日志

### v1.0.1 (2026-04-18)

#### 🔒 安全修复
- **数据外泄风险修复**: 移除对不可信第三方服务（nitter等）的依赖，仅使用Twitter官方API
- **供应链风险修复**: 移除subprocess执行外部命令，避免恶意代码注入
- **权限提升风险修复**: 使用飞书官方SDK代替subprocess跨Skill调用

#### ✨ 新功能
- **一键安装脚本**: `quick_setup.py` 帮助新手快速完成配置
- **错误通知系统**: `notifier.py` 支持飞书Webhook、自定义Webhook、本地日志多渠道通知
- **B站登录态教程**: 详细图文教程帮助用户获取SESSDATA

#### 📚 文档完善
- 新增22条常见问题FAQ
- 新增B站登录态获取教程
- 优化新手引导流程

## 技术特点

### 1. 完全免费 💰
- 使用OpenAI开源Whisper模型
- 无API费用，本地运行
- 只需要一台电脑

### 2. 准确度高 🎯
- 支持tiny/base/small/medium模型
- 长音频自动分段处理
- 优先提取官方字幕

### 3. 自动化 ⚡
- 每日自动检查更新
- 自动转录并同步飞书
- 每周汇总报告

### 4. 可扩展 🔧
- 支持自定义博主列表
- 灵活的定时任务配置
- 完善的错误重试机制

---

## 使用场景

1. **知识工作者**：追踪行业播客、视频，建立个人知识库
2. **内容创作者**：研究竞品内容，提取灵感
3. **研究者**：收集多平台信息，辅助研究
4. **学习者**：将听过的内容变成可搜索的文字

## 快速开始

```bash
# 1. 安装依赖
pip install -r .skills/content-monitor/scripts/requirements.txt

# 2. 配置博主列表
# 编辑 ./内容监控/配置/博主列表.md

# 3. 运行监控
python .skills/content-monitor/scripts/monitor.py
```
