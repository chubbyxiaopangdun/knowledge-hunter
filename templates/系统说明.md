# 内容监控系统说明

## 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    主监控脚本 (monitor.py)              │
│                    定时执行 (每天8:00)                   │
└─────────────────────┬───────────────────────────────────┘
                      │
          ┌───────────┼───────────┬────────────┐
          ▼           ▼           ▼            ▼
    ┌──────────┐ ┌──────────┐ ┌────────┐ ┌──────────┐
    │ 小宇宙    │ │ YouTube  │ │  B站   │ │ Twitter │
    │ xiaoyuzhou│ │ youtube  │ │bilibili│ │ twitter │
    └────┬─────┘ └────┬─────┘ └───┬────┘ └────┬─────┘
         │            │           │            │
         └────────────┴─────┬─────┴────────────┘
                            ▼
                    ┌───────────────┐
                    │  语音转文字   │
                    │   (Whisper)   │
                    └───────┬───────┘
                            ▼
                    ┌───────────────┐
                    │ Markdown文件  │
                    │  (带时间轴)   │
                    └───────┬───────┘
                            ▼
                    ┌───────────────┐
                    │  飞书同步     │
                    │ sync_to_feishu│
                    └───────────────┘
```

## 技术方案

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| 语音转文字 | Whisper (base模型) | 开源免费，CPU可运行 |
| YouTube下载 | yt-dlp | 优先提取字幕 |
| B站下载 | yt-dlp | 支持CC字幕提取 |
| RSS解析 | feedparser | 跨平台播客订阅 |
| 数据存储 | 飞书知识库 | 免费云存储 |

## Whisper模型选择

| 模型 | 参数量 | CPU速度 | 推荐场景 |
|------|--------|---------|----------|
| tiny | ~39M | 快速 | 测试/快速预览 |
| base | ~74M | 较快 | **日常使用推荐** |
| small | ~244M | 中等 | 高质量需求 |
| medium | ~769M | 较慢 | 最高质量 |

## 使用流程

1. **首次使用**：安装依赖
   ```bash
   bash ./内容监控/scripts/setup.sh
   ```

2. **添加博主**：编辑 `配置/博主列表.md`

3. **手动执行**：
   ```bash
   python3 ./内容监控/scripts/monitor.py
   ```

4. **自动执行**：每天8:00自动运行（需配置定时任务）

## 注意事项

1. 首次运行Whisper会自动下载模型（约74MB for base）
2. YouTube/B站下载需要网络连接
3. Twitter爬虫可能需要登录态或API访问
4. 飞书同步使用lark-cli工具
