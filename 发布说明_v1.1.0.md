# 知识猎手 v1.1.0 版本更新说明

> 发布日期：2026-04-18

## 核心变化

从**被动存储** → **主动推送价值**

不再只是转录和存储内容，而是主动提取观点、评分、检测冲突，生成每日简报推送给你。

---

## 新增功能

### 1. 观点提取器 (`scripts/viewpoint_extractor.py`)

**功能：**
- 从转录稿提取核心观点（3-5条）
- 提取内容亮点/金句
- 自动判断内容类型（干货/观点/访谈/资讯）

**技术：**
- 支持多LLM提供商（Coze/OpenAI/DeepSeek）
- 本地缓存，节省API调用
- 降级方案：规则提取

**配置：** `配置/LLM配置.json`

---

### 2. 内容评分器 (`scripts/content_scorer.py`)

**评分维度：**
| 维度 | 权重 | 说明 |
|------|------|------|
| 观点质量 | 40% | 观点清晰度、案例支撑、深度分析 |
| 信息密度 | 30% | 无废话、数据支撑、结构清晰 |
| 原创性 | 30% | 新见解、新数据、独特视角 |

**推荐等级：**
- ⭐⭐⭐⭐⭐ ≥8分 → 强烈推荐
- ⭐⭐⭐⭐ ≥7分 → 推荐
- ⭐⭐⭐ ≥6分 → 可以看

**配置：** `配置/评分规则.json`（可自定义权重和阈值）

---

### 3. 冲突检测器 (`scripts/conflict_detector.py`)

**功能：**
- 维护观点数据库
- 检测不同来源的观点冲突
- 生成"谁说了什么 vs 谁说了不同的"格式

**冲突类型：**
- 矛盾型（会vs不会、能vs不能）
- 数值型（增长vs下降、多vs少）
- 优先级型（产品vs运营、技术vs市场）

---

### 4. 简报生成器 (`scripts/briefing_generator.py`)

**简报格式：**
```
📰 今日内容简报 | 2026-04-18

🎬 值得看的视频（3个）
1️⃣ 【标题】by 作者
   核心观点：...
   推荐理由：...

🎧 值得听的播客（2个）
1️⃣ 【标题】by 主播
   核心观点：...
   适合场景：...

💡 观点冲突
「主题」
🔴 A说：...
🔵 B说：...
```

**模板：** `templates/每日简报模板.md`（可自定义）

---

### 5. 定时推送任务 (`scripts/daily_briefing.py`)

**功能：**
- 每天9点自动执行
- 调用上述所有模块
- 推送到飞书

**配置：** `配置/简报配置.json`

## 使用方式

### 启用每日简报

1. 复制配置示例：
   ```bash
   cp 配置/LLM配置示例.json 配置/LLM配置.json
   cp 配置/简报配置示例.json 配置/简报配置.json
   ```

2. 编辑配置文件，填入API Key

3. 告诉诺诺："每天9点给我推送内容简报"

### 手动生成简报

```bash
cd .skills/content-monitor/scripts
python daily_briefing.py --date 2026-04-18
```

## 技术细节

### LLM调用
- 默认使用Coze API
- 支持降级：规则提取（无需API）
- 缓存机制：避免重复调用

### 评分规则
- 完全可配置
- 权重可调整
- 阈值可自定义

### 冲突检测
- 基于关键词匹配
- 支持同义词扩展
- 观点库持久化存储

---

## 已知限制

1. 观点提取依赖LLM质量，可能有误差
2. 冲突检测基于规则，可能漏检
3. 首次运行需要配置LLM
4. 飞书推送需要Webhook配置
