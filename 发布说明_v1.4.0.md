## 📌 版本概述

本次更新从「监控工具」升级为「素材工具」，新增**金句提取器**功能。

**核心升级点**：自动从转录内容中提取金句，存入素材库供内容创作使用。

---

## ✨ 新增功能

### 1. 金句提取器 (`quote_extractor.py`)

**功能**：调用AI从转录文本中提取金句

**特点**：
- 每段内容提取约20个金句
- 自动打主题标签（创业/职场/AI等）
- 记录来源信息（标题、作者、链接、平台）
- 可选的使用建议提示
- 使用 `gpt-4o-mini` 模型，成本优化

**金句格式示例**：
```markdown
> 做产品最重要的不是功能多，而是解决用户真实的问题

- **主题**: 产品
- **来源**: [视频标题](https://example.com) - 作者名
- **平台**: YouTube
```

### 2. 素材库管理器 (`quote_library.py`)

**功能**：管理金句素材库

**特点**：
- 按主题自动分类存储
- 支持按关键词搜索
- 支持按时间排序
- Markdown格式存储，方便复制使用
- 提供统计和导出功能

**素材库目录结构**：
```
quotes/
├── all_quotes.md          # 全部金句汇总
├── by_topic/              # 按主题分类
│   ├── AI创业.md
│   ├── 个人成长.md
│   └── ...
└── metadata.json          # 元数据索引
```

### 3. 监控流程集成

在原有监控流程中自动调用金句提取：

```
内容转录 → AI自动提取约20个金句 → 存入素材库 → 用户创作时提取使用
```

## 🔧 配置说明

### 环境变量配置

```bash
# 是否启用金句提取功能（默认关闭，向后兼容）
ENABLE_QUOTE_EXTRACTION=false

# 每个视频提取的金句数量
QUOTES_PER_VIDEO=20

# 金句提取使用的AI模型（默认gpt-4o-mini，性价比高）
QUOTE_MODEL=gpt-4o-mini

# 素材库存放目录
QUOTES_OUTPUT_DIR=./quotes
```

### 启用金句提取

```bash
# 方式1：设置环境变量
export ENABLE_QUOTE_EXTRACTION=true

# 方式2：创建.env文件
echo "ENABLE_QUOTE_EXTRACTION=true" >> .env
```

## 💡 使用示例

### 命令行直接提取金句

```bash
python scripts/quote_extractor.py \
  --file 输出/某视频.txt \
  --title "视频标题" \
  --author "作者名" \
  --output quotes.md
```

### 素材库管理

```bash
# 查看统计
python scripts/quote_library.py stats

# 按主题列出
python scripts/quote_library.py list --topic AI

# 搜索关键词
python scripts/quote_library.py search "创业"

# 导出全部素材
python scripts/quote_library.py export --output my_quotes.md
```

## ⚠️ 注意事项

1. **向后兼容**：金句提取功能默认关闭，不影响现有工作流程
2. **成本考虑**：使用AI模型提取金句，会产生API费用，建议使用 `gpt-4o-mini`
3. **启用方式**：需设置 `ENABLE_QUOTE_EXTRACTION=true` 启用
4. **素材积累**：金句会持续积累，建议定期整理和清理

## 🚀 升级指南

### 从旧版本升级

1. 拉取最新代码：
   ```bash
   cd .skills/content-monitor
   git pull origin main
   ```

2. 更新依赖：
   ```bash
   pip install -r scripts/requirements.txt
   ```

3. 复制新的环境变量配置：
   ```bash
   # 追加新的配置项到.env
   echo "ENABLE_QUOTE_EXTRACTION=false" >> .env
   ```

   ```bash
   # 编辑.env，将ENABLE_QUOTE_EXTRACTION改为true
   sed -i 's/ENABLE_QUOTE_EXTRACTION=false/ENABLE_QUOTE_EXTRACTION=true/' .env
   ```
