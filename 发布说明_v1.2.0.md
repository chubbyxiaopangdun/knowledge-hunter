## 核心升级：从「内容消费」到「内容创作辅助」

### 解决的问题

> 内容创作者知道要做内容，但不知道写什么

### 解决方案

通过分析监控内容，自动发现热点、生成选题、追踪更新，让创作者不再为选题发愁。

---

## 新增功能

### 1. 话题聚类器 (`scripts/topic_cluster.py`)

**功能**：
- 分析本周监控的所有内容
- 按话题聚类，识别热点
- 计算热度分数 = 讨论量(40%) + 时效性(30%) + 互动(30%)
- 输出：Top 3-5 热点话题

**核心特性**：
- 关键词提取 + 相似度匹配
- 历史数据对比（计算增长率）
- 支持配置话题关键词库

### 2. 选题生成器 (`scripts/topic_generator.py`)

**功能**：
- 基于热点话题生成选题建议
- 基于观点对比生成选题建议
- 生成角度建议（从哪个方向切入）
- 输出：具体选题标题 + 切入角度 + 关键要点

**核心特性**：
- 多种标题模板（警告型/真相型/辩论型/指南型/故事型）
- 可选 LLM 生成（需配置）
- 平台适配建议

### 3. 内容关联器 (`scripts/content_relate.py`)

**功能**：
- 维护用户的"历史选题库"
- 追踪用户选题的相关内容更新
- 发现可以做"追踪/续集"的机会
- 输出：你的选题有新更新

**核心特性**：
- 关键词匹配计算相关性
- 更新类型分类（新增/观点/案例/数据）
- 自动生成追踪建议

### 4. 选题报告生成器 (`scripts/topic_report.py`)

**功能**：
- 组装周报格式的选题报告
- 包含：热点话题、观点对比、选题追踪、选题建议
- 可配置推送时间（默认周日20:00）
- 输出到飞书/本地

## 技术细节

### 热度分数计算

```
热度分数 = 讨论量分 * 40% + 时效性分 * 30% + 互动分 * 30%

讨论量分 = min(内容数量 / 20, 1.0)
时效性分 = 最近7天内容占比
互动分 = (点赞 + 评论*2 + 分享*3) / 浏览量
```

### 话题聚类算法

1. 关键词提取（预定义热点词库 + TF-IDF）
2. 关键词匹配聚合
3. 相似度计算（可选）
4. 按热度排序

### 选题生成模板

| 类型 | 模板 | 示例 |
|------|------|------|
| 警告型 | {topic}的{n}个陷阱 | AI创业的3个陷阱 |
| 真相型 | 关于{topic}的真相 | 关于AI创业的真相 |
| 辩论型 | {topic}A还是B | AI创业A还是B |
| 指南型 | {topic}入门指南 | AI创业入门指南 |
| 故事型 | 我做{topic}的第N天 | 我做AI创业的第30天 |
