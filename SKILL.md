# 知识猎手 (Knowledge Hunter)

> 主动出击，精准捕获有价值的内容，转化为你的知识资产 🎯

## 技能概述

自动监控多平台内容更新，将音频/视频转成文字稿，同步到飞书知识库。

**支持平台：**
- 小宇宙（播客）
- YouTube（视频）
- 哔哩哔哩（视频）
- Twitter（推文）

**核心功能：**
- ✅ 每日自动检查更新
- ✅ 音频/视频转文字稿（Whisper免费转录）
- ✅ 保留时间轴
- ✅ 自动同步到飞书知识库
- ✅ 每周汇总报告

**技术栈（全免费）：**
- Whisper开源模型（本地运行）
- RSS订阅 + API监控
- 飞书CLI同步

## 转录质量说明

### Whisper模型选择

| 模型 | 大小 | 速度 | 准确度 | 推荐场景 |
|------|------|------|--------|---------|
| tiny | 39MB | 最快 | 一般 | 快速预览、测试 |
| base | 74MB | 快 | 较好 | 日常使用（默认） |
| small | 244MB | 中等 | 好 | 高质量需求 |
| medium | 769MB | 慢 | 很好 | 专业场景 |

### 转录时间估算

- **5分钟音频** → 约1分钟转录
- **30分钟音频** → 约5分钟转录
- **120分钟音频** → 约20分钟转录

### 质量优化建议

1. **优先使用官方字幕**：YouTube/B站很多视频有CC字幕，质量最高
2. **使用更大的模型**：base > tiny，small > base
3. **分段处理**：超长音频自动分段，避免内存溢出

## 常见问题

### Q1: Whisper转录很慢怎么办？
A: 使用tiny模型更快，或等待后台任务完成（会自动通知）

### Q2: 转录质量不好怎么办？
A: 优先使用有官方字幕的视频源，或升级到small模型

### Q3: 如何批量添加博主？
A: 提供链接列表，告诉诺诺"批量添加这些博主到监控列表"

### Q4: 飞书同步失败怎么办？
A: 检查授权状态，重新授权：`lark-cli auth login --domain drive,wiki,docs`

### Q5: 如何暂停某个博主的监控？
A: 在博主列表中添加 `# 已暂停` 注释该行即可

## 技术原理

### 小宇宙监控流程
```
RSS订阅 → 解析新节目 → 下载音频 → Whisper转录 → 格式化 → 飞书同步
```

### YouTube/B站监控流程
```
频道监控 → 检测新视频 → 提取字幕/下载音频 → 转录 → 格式化 → 飞书同步
```

### Twitter监控流程
```
用户监控 → 获取新推文 → 文本提取 → 格式化 → 飞书同步
```
