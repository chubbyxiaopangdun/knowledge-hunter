# 知识猎手 (Knowledge Hunter)

> 主动出击，精准捕获有价值的内容，转化为你的知识资产 🎯

**版本**: v1.1.0  
**更新时间**: 2026-04-18  
**状态**: 正式版 (stable)

## 技能概述

自动监控多平台内容更新，将音频/视频转成文字稿，并生成**每日简报推送**。

**支持平台：**
- 小宇宙（播客）
- YouTube（视频）
- 哔哩哔哩（视频）
- Twitter（推文）

**核心功能：**

### v1.1.0 新增：每日简报推送 🚀

| 模块 | 功能 | 说明 |
|------|------|------|
| 观点提取器 | `viewpoint_extractor.py` | 从转录稿提取核心观点（3-5条）、金句亮点、判断内容类型 |
| 内容评分器 | `content_scorer.py` | 基于观点质量、信息密度、原创性打分（0-10），标记值得看的内容 |
| 冲突检测器 | `conflict_detector.py` | 对比近期内容，发现观点冲突，生成"谁说了什么"格式 |
| 简报生成器 | `briefing_generator.py` | 组装每日简报格式（3视频+2播客+1冲突） |
| 定时推送 | `daily_briefing.py` | 每天9点执行，推送到飞书 |

### 基础功能

- ✅ 每日自动检查更新
- ✅ 音频/视频转文字稿（Whisper免费转录）
- ✅ 保留时间轴
- ✅ 自动同步到飞书知识库
- ✅ **观点自动提取（v1.1.0）**
- ✅ **每日简报推送（v1.1.0）**

**技术栈：**
- Whisper开源模型（本地运行）
- RSS订阅 + API监控
- 飞书CLI同步
- **LLM API（观点提取，可选Coze/OpenAI/DeepSeek）**

## 快速开始

### 1. 初始化配置

```bash
# 编辑博主列表
./内容监控/配置/博主列表.md
```

### 2. 添加监控目标

在博主列表中添加你要监控的内容源：

**小宇宙播客：**
```markdown
| 名称 | RSS地址 | 备注 |
|------|---------|------|
| 给女孩的商业第一课 | https://feed.xyzfm.space/xxx | 商业类 |
```

**YouTube频道：**
```markdown
| 名称 | 频道ID | 备注 |
|------|--------|------|
| Ali Abdaal | UCxxxxxxx | 生产力 |
```

**B站UP主：**
```markdown
| 名称 | UID | 备注 |
|------|-----|------|
| 影视飓风 | 12345678 | 科技 |
```

### 3. 执行监控

告诉诺诺：
- "执行内容监控系统"
- "帮我检查小宇宙/YouTube/B站的更新"
- "创建每日内容监控任务"
- **"每天9点给我推送内容简报"** ← v1.1.0新功能

### 4. LLM配置（可选）

创建 `配置/LLM配置.json` 启用观点提取：

```json
{
  "provider": "coze",
  "model": "gpt-4o-mini",
  "temperature": 0.7
}
```

支持提供商：coze / deepseek / openai

## 简报功能详解

### 观点提取器

使用LLM从转录稿中提取：
- **核心观点**（3-5条，每条50字内）
- **内容亮点/金句**（2-3条）
- **内容类型**（干货/观点/访谈/资讯）

### 内容评分规则

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 观点质量 | 40% | 有明确观点、有案例支撑、有深度分析 |
| 信息密度 | 30% | 无废话、有数据支撑、结构清晰 |
| 原创性 | 30% | 新见解、新数据、独特视角 |

**推荐阈值：**
- ⭐⭐⭐⭐⭐ ≥8.0分 → 强烈推荐
- ⭐⭐⭐⭐ ≥7.0分 → 推荐
- ⭐⭐⭐ ≥6.0分 → 可以看

### 冲突检测

自动检测不同博主的观点冲突：
- 矛盾型（会vs不会）
- 数值型（增长vs下降）
- 优先级型（产品vs运营）
